# SublimeWenyan

[中文README](Chinese.md)
![](wenyan-st.png)


## Usage
1. Build wenyan and add is to PATH, so that you can call it anywhere. Do not change the name of the binary files, just keep the platform suffix.

2. Install this plugin so that you can use it in SublimeText.
